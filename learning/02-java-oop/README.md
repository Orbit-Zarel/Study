# 02 Java 面向对象基础

如果说基础语法是在学“句子怎么写”，那面向对象就是在学“怎么组织一个更像样的程序”。

## 这一模块重点理解什么

- 类和对象分别是什么
- 属性和方法是什么
- 构造方法用来做什么
- `this` 表示什么
- 为什么说“对象是在描述现实中的一个东西”

## 1. 类和对象

你可以先这么理解：

- 类：图纸、模板
- 对象：按图纸创建出来的具体实例

例如：

- `Student` 是类
- `new Student("Tom", 18)` 创建出来的是对象

## 2. 属性与方法

属性表示“这个对象有什么数据”。

例如：

- 姓名
- 年龄

方法表示“这个对象会做什么”。

例如：

- 自我介绍
- 学习

## 3. 构造方法

构造方法用于“创建对象时，顺便把初始数据准备好”。

```java
public Student(String name, int age) {
    this.name = name;
    this.age = age;
}
```

这里的 `this.name = name;` 可以理解成：

- 左边的 `this.name` 是对象自己的属性
- 右边的 `name` 是传进来的参数

## 4. 封装

封装可以先简单理解为：

- 对象把自己的数据和行为放在一起
- 外部尽量通过方法和对象交互，而不是随意修改内部细节

后面学到 `private`、`getter/setter` 时会更完整。

## 5. 学完这一模块后你应该能做到

- 看懂一个最基础的 Java 类
- 自己写一个类和构造方法
- 创建对象并调用它的方法
